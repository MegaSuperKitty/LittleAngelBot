# Report Format

## Suggested Structure

- Title page
- Executive summary
- Background / context
- Findings (bullets or numbered lists)
- Analysis
- Recommendations
- Appendix (optional)

## Markdown Tips

- Use `#` and `##` for section headings.
- Use lists for key points.
- Keep paragraphs short for readability.
- Add inline citations like `[1]` and include a `## 参考资料` section.
