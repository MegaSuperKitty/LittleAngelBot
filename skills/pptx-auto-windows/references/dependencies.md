# Dependencies

## Required (Path A)

- `python-pptx`

Install:

```bash
python -m pip install python-pptx
```

## Optional (Path B: HTML -> PDF)

- `weasyprint` (pure-Python API, but requires system libraries on Windows)

Install:

```bash
python -m pip install weasyprint
```

If `weasyprint` is not available, skip Path B or use an already-installed HTML-to-PDF tool.
